### *Сервис автоматической генерации описаний товаров для маркетплейсов*

Что делает пользователь: загружает фото заводской этикетки. Сервис берет это изображение, считывает текст с этикетки. В случае, если на этикетке отсутствует название товара - можно вписать его вручную. Далее генерируется готовое описание для карточки товара.
