### *Сервис распознавания текста с этикеток товаров*

Что делает пользователь: загружает фото заводской этикетки. Сервис извлекает весь текст с этикетки (название, состав, инструкции, производителя). Если название товара не читается автоматически — можно добавить его вручную. Полученный текст готов для копирования в карточку товара.
