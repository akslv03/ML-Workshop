Для дальнейшей работы не забудьте выполнить следующую команду в терминале:

docker exec -it product-description-ollama ollama pull qwen3.5:0.8b

Эта команда войдет в контейнер product-description-ollama и загрузит модель qwen3.5:0.8b.
После этого вы сможете использовать модель qwen3.5:0.8b в своем проекте.
